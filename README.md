# Backend-Server
Este es un pequeño ejemplo de un servidor backend desarrollado me JavaScript con Express y usando como base de datos a MongoDB

# Instalación
Ejecutar el comando:
```
npm install
```
Para instalar las dependencias del proyecto
# Ejecución
Ejecute el comando:
```
npm start
```
Para levantar el servidor

## Extra
Dentro del proyecto existe otro proyecto dentro de la carpeta
Google-SingIn que es un pequeño ejemplo del uso de la autenticación de Google en un frontend básico pero funcional
